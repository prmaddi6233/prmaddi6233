## 🔭 What I Do

I design and operate AWS platforms (**100+ accounts**) that are repeatable, policy-aligned, observable, and cost-aware from the start. My focus: making the secure, cost-efficient path the *easiest* path for product teams — without slowing delivery.

**Currently building** → [**cloud-finops-agent**](https://github.com/prmaddi6233/cloud-finops-agent) — an open-source tool that validates cloud cost optimizations with evidence before anyone touches production. Deterministic checks, real production metrics, and canary deployments with automated rollback.
