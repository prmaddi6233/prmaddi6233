## 👋 About Me

- 🏗️ I run platform engineering for a **100+ account AWS Organization** — account lifecycle automation, security baselines, and FinOps analytics
- 💡 My thesis: **cost recommendations rot in backlogs because nobody proves they're safe** — so I build the evidence layer: deterministic checks, production metrics, canary validation
- 🔭 Currently building [**cloud-finops-agent**](https://github.com/prmaddi6233/cloud-finops-agent) — tiered validation with production canaries and automated rollback
