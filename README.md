> 🔥 **Currently building** [cloud-finops-agent](https://github.com/prmaddi6233/cloud-finops-agent) — tiered FinOps validation with production canaries, SARIF findings, and fail-closed safety.

**Engineering principles I live by:**

| Principle | Meaning |
|-----------|--------|
| Document *why not*, not just *how* | Rejected alternatives prevent repeated mistakes |
| Policy engines over runbooks | SCPs and Kyverno enforce — docs get forgotten |
| Fail closed or it's not safety | If the system can't verify, it stops |
